# Cat vs Dog Image Classifier

A Python Machine Learning project deployed on Streamlit Cloud that classifies images as either a **Cat** or a **Dog** using a trained Logistic Regression model.

## Project Structure

```text
project/
├── app.py
├── cat_dog_model.pkl
├── requirements.txt
└── README.md
```

## Running Locally

1. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```
